# Leaderboard Guild Resource

Returns a list of top 10 guilds based on `{category}`.

```sh
/v1/leaderboard/guilds/{category}
```

## Available Category

- `level`
- `exp`

## Rate Limit

The rate limit for the player statistics is 750 requests per 30 minutes per IP.
