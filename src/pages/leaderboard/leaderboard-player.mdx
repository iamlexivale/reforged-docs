# Leaderboard Player Resource

Returns a list of top 10 players based on `{category}`.

```sh
https://api.reforged.world/v1/leaderboard/players/{category}
```

## Response

```json
{}
```

### `?` object

| Field | Type | Description | Options |
| ----- | ---- | ----------- | ------- |
| ?     | ?    | ?           |         |

## Rate Limit

The rate limit for the player statistics is 750 requests per 30 minutes per IP.
