# Leaderboard Nation Resource

Returns a list of top 10 nations based on coins.

```sh
/v1/leaderboard/nations
```

## Response

```json
{
    "nations": [
        {
            "nation": String,
            "coins": Number
        },
        // ...
    ],
    "request": {
        "timestamp": Date,
        "version": Number
    }
}
```

### `nations` object

| Field  | Type   | Description         | Options |
| ------ | ------ | ------------------- | ------- |
| nation | String | Return nation name  |         |
| coins  | Number | Return nation coins |         |

## Rate Limit

The rate limit for the player statistics is 750 requests per 30 minutes per IP.
