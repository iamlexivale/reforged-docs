# Guild List Resource

Returns a list of all guilds.

```sh
/v1/guilds
```

## Response

```json
{
    "guilds": [
        {
            "id": String,
            "name": String,
            "prefix": String,
            "guildMaster": {
                "uuid": String,
                "role": {
                    "level": Number
                },
                "joinDate": Date,
                "lastLogin": Date
            },
            "home": {
                "world": String,
                "x": Number,
                "y": Number,
                "z": Number,
                "yaw": Number,
                "pitch": Number
            },
            "status": String,
            "tier": {
                "level": Number
            },
            "balance": Number,
            "members": [
                {
                    "uuid": String,
                    "role": {
                        "level": Number
                    },
                    "joinDate": Date,
                    "lastLogin": Date
                }
                // ...
            ],
            "invitedMembers": Object,
            "allies": Object,
            "pendingAllies": Object,
            "codes": Object,
            "vaults": Object,
            "lastDefended": Number,
            "creationDate": Date
        }
        // ...
    ],
    "request": {
        "timestamp": Date,
        "version": Number
    }
}
```

### `guildMaster` object

| Field     | Type   | Description              | Options |
| --------- | ------ | ------------------------ | ------- |
| uuid      | String | Return guild master uuid |         |
| role      | Object | Return role in number    | `level` |
| joinDate  | Date   | Return join date         |         |
| lastLogin | Date   | Return last login        |         |

### `guildMaster.role` object

| Field | Type   | Description           | Options |
| ----- | ------ | --------------------- | ------- |
| level | Number | Return role in number |         |

### `home` object

| Field | Type   | Description           | Options |
| ----- | ------ | --------------------- | ------- |
| world | String | Return world name     |         |
| x     | Number | Return coordinate x   |         |
| y     | Number | Return coordinate y   |         |
| z     | Number | Return coordinate z   |         |
| yaw   | Number | Return yaw rotation   |         |
| pitch | Number | Return pitch rotation |         |

### `tier` object

| Field | Type   | Description  | Options |
| ----- | ------ | ------------ | ------- |
| level | Number | Return level |         |

### `members` object

| Field     | Type   | Description              | Options |
