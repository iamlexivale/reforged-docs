# Player List Resource

Returns a list of all players.

```sh
/v1/players
```

## Response

```json
{
    "players": [
        {
            "username": String,
            "uuid": String,
            "lastlogin": Number
        },
        // ...
    ],
    "request": {
        "timestamp": Date,
        "version": Number
    }
}
```

### `players` object

| Field     | Type   | Description                | Options                         |
| --------- | ------ | -------------------------- | ------------------------------- |
| username  | String | return player username     |                                 |
| uuid      | String | return player uuid         |                                 |
| lastlogin | Number | return player latest login | _can be convert to date format_ |

## Rate Limit

The rate limit for the player statistics is 750 requests per 30 minutes per IP.
