# Profile Statistics Resource

Returns all data profile information based on username and profile uuid.

```sh
/v1/player/{username}/{profile}
```

## Response

```json
{
    "player": {
        "username": String,
        "uuid": String,
        "lastlogin": Number,
        "profile": {
            "name": String,
            "uuid": String,
            "health": Number,
            "exp": Number,
            "level": Number,
            "balance": Number,
            "class": String,
            "attributes": {
                "dexterity": Number,
                "luck": Number,
                "strength": Number,
                "vitality": Number,
                "intelligence": Number
            },
            "professions": {
                "smithing": { "exp": Number, "level": Number },
                "enchanting": { "exp": Number, "level": Number },
                "woodcutting": { "exp": Number, "level": Number },
                "fishing": { "exp": Number, "level": Number },
                "farming": { "exp": Number, "level": Number },
                "alchemy": { "exp": Number, "level": Number },
                "mining": { "exp": Number, "level": Number },
                "smelting": { "exp": Number, "level": Number }
            }
        }
    },
    "request": {
        "timestamp": Date,
        "version": Number
    }
}
```

## Rate Limit

The rate limit for the player statistics is 750 requests per 30 minutes per IP.
