# Player Profile Resource

Returns tiny data profile information based on username.

```sh
/v1/player/{username}
```

## Response

```json
{
    "player": {
        "username": String,
        "uuid": String,
        "primary_group": String,
        "profiles": [
            {
                "name": String,
                "uuid": String,
                "health": Number,
                "exp": Number,
                "level": Number,
                "balance": Number,
                "class": String
            },
            // ...
        ]
    },
    "request": {
        "timestamp": Date,
        "version": Number
    }
}
```

### `profiles` object

| Field   | Type   | Description            | Options |
| ------- | ------ | ---------------------- | ------- |
| name    | String | Return profile names   |         |
| uuid    | String | Return profile uuid    |         |
| health  | Number | Return profile health  |         |
| exp     | Number | Return profile exp     |         |
| level   | Number | Return profile level   |         |
| balance | Number | Return profile balance |         |
| class   | String | Return profile class   |         |

## Rate Limit

The rate limit for the player statistics is 750 requests per 30 minutes per IP.
